**ModbusX** is a cross-platform application for managing Modbus TCP and RTU connections. Built with Tauri, Rust, and React/TypeScript, it supports multiple simultaneous connections, providing a modern and efficient interface for industrial communication.

---

## Key Features

- **Multi-Protocol Support**: Manage both Modbus TCP and RTU connections.
- **Simultaneous Connections**: Handle multiple connections at the same time.
- **Cross-Platform**: Available for macOS (Intel & ARM), Windows, and Linux.
- **Modern UI**: Built with React and TailwindCSS for a clean and responsive interface.
- **Customizable Settings**: Easily configure connection parameters and preferences.
- **Lightweight**: Powered by Tauri, ensuring minimal resource usage.

## Installation

Pre-built binaries are available in the [Releases](https://github.com/KonorOko/modbusx/releases) section of this repository.

### macOS

1. Download the `.dmg` file for your architecture (Intel or ARM) from the [Releases](https://github.com/KonorOko/modbusx/releases) page.
2. Open the `.dmg` file and drag the app to your `Applications` folder.
3. Launch the app from your Applications folder.

### Windows

1. Download the `.exe` installer from the [Releases](https://github.com/KonorOko/modbusx/releases) page.
2. Run the installer and follow the on-screen instructions.
3. Launch the app from the Start Menu.

### Linux

1. Download the appropriate binary for your distribution from the [Releases](https://github.com/KonorOko/modbusx/releases) page.
2. Make the binary executable:
   ```bash
   chmod +x modbusx
   ```
3. Run the application:
   ```bash
   ./modbusx
   ```
